## 最终最优配置 (R19/R29)

### 模型参数

```python
# VMD
VMD_K优化: range(2, 7), freq_ratio=1.5, alpha=2000
VMD分解: 仅训练集, 趋势线性外推, 模态persistence

# LSTM架构
MultiFeatureLSTM: input=5, hidden=6, 2-layer, dropout=0.25
                  双层FC: hidden→hidden/2→1 + ReLU
SingleFeatureLSTM: hidden=4, 2-layer, dropout=0.25
                   双层FC: hidden→hidden/2→1 + ReLU

# LSTM训练
optimizer: Adam(lr=0.002, wd=1e-4)
scheduler: ReduceLROnPlateau(factor=0.5, patience=15, min_lr=1e-5)
epochs: 1000, patience: 60, loss: MSELoss
seq_len: 6, stride: 1 (42 samples)

# CatBoost
基线: iterations=1500, depth=6, lr=0.02, l2=3, 全8维特征
融合: iterations=1500, depth=6, lr=0.015, l2=2

# 特征工程
8维: 4原始因子 + lag1 + lag2 + month_sin + month_cos
```

## 关键发现

### 正向改动（已保留）

| 排名 | 改动 | 避雷器增益 | 说明 |
|:---:|------|:---:|------|
| 1 | **双层FC头** (hidden→hidden/2→1+ReLU) | +0.067 | 最大单项贡献, 增强LSTM特征提取 |
| 2 | **K=2-7优化** (原K=3-7) | +0.055 | K=2对简单信号更合适 |
| 3 | **hidden缩小** (8/6→6/4) | +0.100 | 42样本下小模型防过拟合 |
| 4 | **CB基线8特征** (原4特征) | +0.011 | CB基线提升传导至融合层 |

### 负向改动（已回退）

| 改动 | 避雷器损失 | 原因 |
|------|:---:|------|
| Per-material VMD alpha | -0.15~0.20 | 破坏频率一致性 |
| K范围扩大(3-10) | -0.27 | 过分解 |
| seq_len≠6 | -0.20~0.28 | 6是月度数据最优窗口 |
| dropout≠0.25 | -0.10~0.19 | 0.25精确平衡 |
| 3层LSTM/FC | -0.05~0.11 | 更深=过拟合 |
| 数据增强(噪声) | -0.05 | 污染已稀缺的训练信号 |
| ELU替代ReLU | -0.17 | ReLU稀疏性更适合小数据 |

### 模型鲁棒性观察

- **VMD K范围**是最敏感参数: K太小(2)→欠分解, K太大(8+)→过分解
- **LSTM hidden size**: 42样本下, hidden=6/4是上限, 更大必过拟合
- **FC深度**: 2层最优, 1层欠拟合, 3层过拟合
- **Dropout**: 0.25是甜点, ±0.05都会显著影响

## 未达0.8的根因分析

### 避雷器 (0.392→0.614)

已提取了当前数据+模型组合的大部分可预测信号。剩余0.386的不可解释方差来自:
1. **年度随机峰型**(40%双峰/30%单5月/30%单8月): 每年模式不同, 48样本下LSTM难以学习3种模式
2. **年度随机基线+噪声**: 每年baseline和noise_std独立随机, 增加不可预测性
3. **48训练样本**: 深度学习的样本量硬限制

### 电缆 (0.341)

VMD-LSTM-CatBoost始终不如纯VMD-CatBoost:
- LSTM预测的IMF精度不如IMF直接外推
- 电缆的多频率信号(年+半年+季度)在48样本下对LSTM过复杂
- **解决方案**: 对电缆使用VMD-CatBoost(不用LSTM)

### 变压器 (负值)

零值从固定3月/年→随机1-2月/年(50%减少):
- 季节模式一致性被破坏, VMD无法稳定分解
- CatBoost基线降为负值(无因子可预测)
- **解决方案**: 恢复变压器冬季固定零值, 或使用周度数据
