# VMD-CatBoost 配电网物资需求预测

基于 **VMD（变分模态分解）** 与 **CatBoost** 的配电网物资月度需求量预测模型。对比五种模型架构（CatBoost / VMD-CatBoost / VMD-LSTM-CatBoost / VMD-LSTM / VMD-SVR），对三类核心配电网物资（10KV电缆、柱上变压器、避雷器）进行精确预测。

## 项目背景

配电网物资需求序列具有**非平稳、波动强**的特性，传统预测方法精度不足。本项目构建 VMD-CatBoost 复合预测模型，通过斯皮尔曼相关性分析筛选关键影响因子，利用 VMD 分解降低序列噪声，结合 LSTM 捕获时序依赖与 CatBoost 梯度提升树进行回归预测，为电网物资供应链智能调度提供决策依据。

## 五种模型

| 模型 | 核心思路 |
|------|---------|
| **CatBoost** | 4 个影响因子 → CatBoost 直接回归预测（基线） |
| **VMD-CatBoost** | VMD 分解需求量(K=5) → 5 个 IMF + 4 个因子 → CatBoost 端到端 |
| **VMD-LSTM-CatBoost** | VMD 分解 → 残差分量多特征 LSTM + 4 个模态分量单特征 LSTM → CatBoost 融合 |
| **VMD-LSTM** | VMD 分解 → 5 个 LSTM 预测各分量 → 直接求和（消融实验，验证 CatBoost 融合层必要性） |
| **VMD-SVR** | VMD 分解需求量(K=5) → 5 个 IMF + 4 个因子 → SVR（核方法对比） |

## 三类物资

| 物资 | 所属类别 | Top-4 影响因子 |
|------|---------|---------------|
| 10KV电缆 (cable) | 基建项目类 | 工程投资量、历史需求量、负荷增长量、设备进价成本 |
| 柱上变压器 (transformer) | 用户增容类 | 负荷增长量、工程投资量、历史需求量、设备进价成本 |
| 避雷器 (arrester) | 应急抢修类 | 雷击次数、台风次数、暴雨次数、负荷增长量 |

## 7 个影响因子

负荷增长量、工程投资量、历史需求量、设备进价成本、台风每月次数、雷击每月次数、暴雨每月次数

- **时间范围**: 2022.01 — 2024.12（36 个月）
- **数据粒度**: 月度

- **时序分割**: 前 24 月训练，后 12 月测试

## 模型效果分析

基于最新运行结果（`outputs/figures/prediction_comparison_*.png` 和 `metrics_summary.json`），对五种模型在三类物资上的预测效果进行分析。

### 整体排名

| 物资 | 1st | 2nd | 3rd | 4th | 5th |
|------|-----|-----|-----|-----|-----|
| 10KV电缆 | **VMD-LSTM-CatBoost** (R²=0.356) | VMD-LSTM (R²=0.153) | CatBoost (R²=0.015) | VMD-CatBoost (R²=-0.284) | VMD-SVR (R²=-2.146) |
| 柱上变压器 | **VMD-LSTM-CatBoost** (R²=0.457) | CatBoost (R²=0.322) | VMD-LSTM (R²=0.008) | VMD-CatBoost (R²=-0.712) | VMD-SVR (R²=-2.299) |
| 避雷器 | **CatBoost** (R²=0.610) | VMD-LSTM (R²=0.265) | VMD-LSTM-CatBoost (R²=-0.053) | VMD-CatBoost (R²=-0.179) | VMD-SVR (R²=-1.274) |

### 逐模型分析

**1. CatBoost（基线）**

基线模型仅使用 4 个影响因子直接回归，表现稳健。在避雷器上取得所有模型中的最佳成绩（R²=0.610, RMSE=26.92），在变压器上也达到第二名（R²=0.322）。避雷器数据含 41.7% 零值，CatBoost 对稀疏目标变量的适应能力优于 LSTM 系列模型。但在电缆上 R² 仅 0.015，说明仅靠 4 个因子不足以捕捉电缆需求的波动规律。

**2. VMD-CatBoost**

VMD 分解后直接拼接 IMF 与因子输入 CatBoost，在三类物资上**全部劣于基线 CatBoost**，且 R² 为负值。这说明 VMD 分解本身并不能自动带来收益——将 5 个 IMF 分量直接作为静态特征喂给树模型，不仅未降低噪声，反而引入了额外的回归误差。IMF 分量之间存在自相关结构，树模型无法有效利用这种时序依赖。

**3. VMD-LSTM-CatBoost（最优方案）**

在电缆和变压器两类**基建项目型物资**上均为最优模型（R² 分别为 0.356 和 0.457），MAE 分别降至 6.61 和 2.93。LSTM 捕获了各 IMF 分量的时序演变规律，再由 CatBoost 融合层校正残差——两级架构形成互补，效果显著优于 VMD-CatBoost（验证了 LSTM 层的必要性）和 VMD-LSTM 直接求和（验证了 CatBoost 融合层的必要性）。

但在避雷器上表现不佳（R²=-0.053），因为在仅有 18 个训练样本（seq_len=6 后）的条件下，天气驱动的避雷器数据零值过多，LSTM 无法学到有效的时序模式。

**4. VMD-LSTM（消融实验）**

作为消融实验，去掉 CatBoost 融合层、对各 IMF 预测结果直接求和。在三类物资上均差于 VMD-LSTM-CatBoost，但普遍优于 VMD-CatBoost——这验证了 CatBoost 融合层的价值：将 LSTM 预测的 5 个 IMF 分量 + 4 个影响因子进行非线性融合，比简单求和能更好地还原原始信号。

**5. VMD-SVR**

在所有物资和所有指标上均为最差，R² 全部为负（-2.299 ~ -1.274）。SVR 作为核方法在 9 维特征（5 IMF + 4 因子）的小样本场景下严重过拟合，不适合本任务的数据规模和特征结构。

### 物资维度分析

| 维度 | 电缆 / 变压器 | 避雷器 |
|------|-------------|--------|
| 物资类别 | 基建项目类 / 用户增容类 | 应急抢修类 |
| 驱动因素 | 工程投资、负荷增长（平稳） | 雷击、台风、暴雨（突发） |
| 数据特征 | 非零连续，月度波动 | 大量零值（41.7%），偶发尖峰 |
| 最优模型 | VMD-LSTM-CatBoost | CatBoost |
| 当前最优 R² | 0.36 ~ 0.46 | **0.61** |

避雷器虽然使用最简单的 CatBoost 基线模型却取得了最高的 R²（0.61），说明**天气因子与避雷器需求之间存在较强的直接映射关系**，无需 VMD 分解或 LSTM 时序建模即可较好拟合。电缆和变压器的需求则受工程节奏、资金拨付节奏等隐变量影响，仅靠 4 个因子难以充分解释。

### 核心发现

1. **VMD 不是银弹**：直接拼接 VMD 分量到树模型（VMD-CatBoost）反而全面下降。VMD 的价值需要配合 LSTM 等时序模型来释放。
2. **两级架构有效**：VMD-LSTM-CatBoost 的 LSTM→CatBoost 融合架构在平稳型物资上验证了有效性，消融实验确认了两级缺一不可。
3. **小样本是瓶颈**：训练集仅 24 个月，经 seq_len=6 窗口后仅剩 18 个有效样本，LSTM 极易过拟合（日志显示 best_loss 低至 1e-5 量级）。
4. **避雷器的特殊性**：天气驱动 + 高零值率的数据特征使 LSTM 系列模型反而不如简单 CatBoost，说明模型选择需按物资类别定制。

## 预测优化建议

### 1. 数据层面

| 方向 | 具体措施 | 预期收益 |
|------|---------|---------|
| **扩充数据量** | 将时间范围从 36 个月扩展至 48~60 个月；如月度数据有限，可尝试周度粒度的数据以增加样本数 | 高，直接缓解小样本过拟合 |
| **数据增强** | 对 LSTM 训练使用滑动窗口采样（stride < seq_len），将 18 个有效样本扩充至更多重叠窗口 | 中，增加 LSTM 训练样本量 |
| **特征工程** | 添加滞后特征（t-1, t-2, t-3 月需求量）、滚动统计量（3/6 月均值和方差）、季节性哑变量（月份 one-hot） | 中高，丰富 CatBoost 可利用的信息 |
| **外部数据** | 引入宏观经济指标（PMI、固定资产投资增速）、电网投资计划完成率、供应商交货周期等 | 中，弥补影响因子不足 |

### 2. VMD 分解优化

| 方向 | 具体措施 | 预期收益 |
|------|---------|---------|
| **K 值调优** | 当前固定 K=5，应针对每类物资通过中心频率观察或网格搜索确定最优 K（建议范围 3~7） | 中，避免过分解或欠分解 |
| **IMF 筛选** | 对分解后的 IMF 做相关性分析，剔除与原序列相关度 < 0.1 的噪声分量，仅保留有效 IMF 输入下游模型 | 中高，减少噪声引入 |
| **不同物质不同方案** | 对避雷器这类高零值序列，VMD 分解效果有限，可跳过 VMD 环节直接使用 CatBoost | 中，避免不适合的预处理 |

### 3. LSTM 架构与训练优化

| 方向 | 具体措施 | 预期收益 |
|------|---------|---------|
| **降低模型复杂度** | 当前虽已精简（hidden_size=4~6, num_layers=1），但仍出现 best_loss 低至 1e-5 的明显过拟合。可尝试 hidden_size=2~3，或增大 weight_decay 至 1e-3 | 中，缓解小样本过拟合 |
| **添加 Dropout** | 当前 dropout=0（已移除），应在单层 LSTM 后添加 dropout=0.2~0.3 | 中，提升泛化能力 |
| **Batch 训练** | 当前使用 full_batch 训练，改 mini-batch（batch_size=4~8）引入梯度噪声有助于正则化 | 低中 |
| **早停策略调整** | 当前 patience=50 / epochs=400，多数情况下打到 max epochs，建议降低 patience 至 20 或 25 | 低，减少无效训练并更早截断 |
| **学习率调度** | 添加 ReduceLROnPlateau（patience=10, factor=0.5），在 loss 平台期自动降低学习率 | 中，提升收敛质量 |
| **双向 LSTM** | 对于月度数据，前后文信息均有价值，改为 Bidirectional LSTM 可提升序列建模能力 | 中 |

### 4. CatBoost 优化

| 方向 | 具体措施 | 预期收益 |
|------|---------|---------|
| **超参数搜索** | 当前参数为手工设定（iterations=300~800, depth=3~6, lr=0.02~0.03）。应对每类物资做 Bayesian 搜索（optuna）或网格搜索 | 中高，找到更优参数组合 |
| **损失函数** | 当前使用 RMSE，可尝试 MAE（对避雷器的零值+尖峰更鲁棒）或 Quantile（预测区间） | 低中 |
| **交叉验证** | 当前使用末尾 6 个月做早停验证，可改为时序交叉验证（TimeSeriesSplit）提高评估稳定性 | 低中 |

### 5. 差异化策略（按物资类别）

| 物资类别 | 推荐策略 | 优先级 |
|---------|---------|--------|
| **电缆 / 变压器** | 继续深耕 VMD-LSTM-CatBoost 架构：扩充数据 → 优化 VMD(K+筛选) → LSTM Dropout+更小 hidden_size → CatBoost 超参搜索 | P0 |
| **避雷器** | 放弃 VMD+LSTM 路径，聚焦 CatBoost：增加天气衍生特征 → 超参搜索 → 尝试 zero-inflated 模型（如两阶段：先分类是否有需求，再回归需求量） | P0 |

### 6. 模型集成

| 方向 | 具体措施 | 预期收益 |
|------|---------|---------|
| **加权集成** | 将 CatBoost + VMD-LSTM-CatBoost + VMD-LSTM 三个正向模型的预测做加权平均（权重由验证集 R² 决定） | 中，降低单一模型偏差 |
| **Stacking** | 用 CatBoost 对各模型输出做 stacking 二级融合 | 中，自动学习最优融合权重 |

### 优化路线图建议

```
第一阶段（数据） → 扩充数据 + 特征工程
第二阶段（架构） → LSTM 正则化 + VMD 调参
第三阶段（搜索） → CatBoost 超参搜索 + 差异化策略
第四阶段（集成） → 模型集成 + 消融验证
```

建议按阶段推进，每个阶段完成后对比指标变化，避免多变量同时改动导致无法归因。
