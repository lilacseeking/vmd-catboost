## 四、核心发现

### 4.1 优化措施效果评估

| 优化措施 | 效果 | 分析 |
|---------|------|------|
| 数据扩充至60月 | 部分有效 | 电缆/变压器 CatBoost R² 提升 (0.015→0.338, 0.322→0.364)，但VMD系列模型未同步受益 |
| 避雷器双峰策略 | **负面** | R² 从 0.610 暴跌至 -0.546。双旺季+双淡季的六段式零值模式过于复杂，超出当前模型能力 |
| VMD K值调优 | 正确运作 | 电缆/变压器维持K=5，避雷器正确降至K=3（检测到频率混叠），但未带来性能提升 |
| IMF相关性筛选 | 未触发 | 所有IMF与原序列相关度均>0.1，无分量被剔除。说明模拟数据的IMF均有信息量 |
| 滑动窗口采样 | 小幅改善 | 训练样本从18→21个 (stride=2)，但LSTM仍严重过拟合 |

### 4.2 与优化前对比

> 注意: 优化前后使用不同数据集（36月 vs 60月），R²不可直接比较。下表中的"变化"反映数据+模型共同变化。

| 物资 | 模型 | 优化前R² (36月) | 优化后R² (60月) | 方向 |
|------|------|:---:|:---:|:---:|
| 电缆 | CatBoost | 0.015 | 0.338 | ↑ 增长趋势被更长的训练序列捕获 |
| 电缆 | VMD-LSTM-CatBoost | 0.356 | -0.079 | ↓ 数据变长但测试期波动更大 |
| 变压器 | CatBoost | 0.322 | 0.364 | → 基本持平 |
| 变压器 | VMD-LSTM-CatBoost | 0.457 | 0.309 | ↓ 略有下降 |
| 避雷器 | CatBoost | **0.610** | -0.546 | ↓↓ 双峰策略导致灾难性退化 |
| 避雷器 | VMD-LSTM-CatBoost | -0.053 | -0.999 | ↓↓ 同上 |

### 4.3 VMD-LSTM-CatBoost 的两级架构仍然有效

在电缆和变压器上，VMD-LSTM-CatBoost的MAE均优于CatBoost（电缆7.63 vs 8.10, 变压器3.03 vs 4.14），说明LSTM→CatBoost融合层在**绝对值预测精度**上有优势，但在R²（趋势拟合度）上不如CatBoost。

消融实验再次验证：VMD-LSTM（直接求和）全面差于VMD-LSTM-CatBoost，CatBoost融合层不可或缺。

### 4.4 避雷器灾难性退化的根因

新策略 `1-3月+7-9月=0` 相比旧策略 `11-3月=0`：

| 维度 | 旧策略 | 新策略 |
|------|--------|--------|
| 零值月份 | 5个月连续 | 6个月分两段 |
| 旺季 | 4-10月 (7个月连续) | 4-6月+10-12月 (6个月分离) |
| 模式复杂度 | 单峰+单谷 | 双峰+双谷 |
| 可预测性 | 高 (周期清晰) | 低 (多段切换) |

双峰双谷模式使测试期12个月内包含了两次零值段切换，模型无法准确捕捉切换边界。

## 五、LSTM 过拟合诊断

所有LSTM模型的best_loss均在 1e-5 ~ 1e-3 量级，且多数达到max_epochs (400)，早停机制完全失效：

| 物资 | 模型类型 | best_loss | 收敛epoch |
|------|---------|-----------|-----------|
| 电缆 | MultiFeature | 7.48e-4 | 400(max) |
| 电缆 | SingleFeature×4 | 8.6e-5 ~ 1.07e-3 | 222~400 |
| 变压器 | MultiFeature | 2.50e-3 | 400(max) |
| 变压器 | SingleFeature×4 | 6.1e-5 ~ 1.52e-3 | 215~400 |
| 避雷器 | MultiFeature | 7.15e-4 | 400(max) |
| 避雷器 | SingleFeature×2 | 4.57e-4 ~ 1.11e-3 | 400(max) |

21个训练样本对LSTM仍严重不足。stride=2虽然从理论上增加了正则化（窗口间不完全重叠），但样本绝对数量仍然太小。

## 六、优化路线图 v2

基于本次运行结果，调整优化优先级：

### P0 — 立即修复

| 措施 | 说明 |
|------|------|
| **回退避雷器策略** | 双峰双谷模式过于复杂。建议保留4-6月+10-12月旺季但**取消7-9月强制零值**，改为低需求区间（如0~20台） |
| **降低 LSTM 复杂度** | hidden_size 从 4~6 降至 2~3；添加 dropout=0.3；weight_decay 从 1e-4 提至 1e-3 |
| **减小滑动窗口重叠** | stride=2时相邻窗口仅差2月，窗口间高度相关。建议 stride=3~4 减少过拟合 |

### P1 — 架构改进

| 措施 | 说明 |
|------|------|
| **CatBoost 超参搜索** | 对每种物资做 optuna Bayesian 搜索 (iterations, depth, lr, l2_leaf_reg) |
| **特征工程** | 添加滞后1~3月需求量、3月/6月滚动均值、月份one-hot哑变量 |
| **早停策略强化** | patience 从 50 降至 20；添加 ReduceLROnPlateau |

### P2 — 差异化策略

| 物资 | 策略 |
|------|------|
| **电缆/变压器** | 聚焦 CatBoost + 特征工程。VMD-LSTM-CatBoost的MAE优势值得保留，但需先解决LSTM过拟合 |
| **避雷器** | 回退策略后，尝试 **two-stage模型**: Stage1分类(是否有需求)→Stage2回归(需求量)。天气因子对分类应有良好区分度 |

### P3 — 探索方向

| 措施 | 说明 |
|------|------|
| **双向LSTM** | Bidirectional LSTM可能更好地捕捉季节转换 |
| **模型集成** | CatBoost + VMD-LSTM-CatBoost 加权平均，利用前者趋势能力+后者精度优势 |
| **周度粒度数据** | 将月度数据拆为周度，可获4倍样本量 (48月→192周) |

VMD K值调优              ██████░░░░  机制正确 (运作但未提升性能)

本次优化验证了**数据扩充**对CatBoost的有效性，暴露了**避雷器策略调整**的风险，并确认了**LSTM在小样本场景下的根本性瓶颈**。下一步应聚焦LSTM正则化 + CatBoost超参搜索 + 避雷器策略修正。
